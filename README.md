Платформа для подготовки к ОГЭ по математике, аналог СДАМ ГИА. Есть авторизация, регистрация с подтверждением через почту, возможность создания пользовательских тестов, статистика пользователя, интеграция ИИ для решения задач. Создал докер для запуска RabbitMq, Celery и Postgres просто для демонтрации функционала.
